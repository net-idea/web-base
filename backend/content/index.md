# Welcome

This is an example content page rendered from Markdown.

## About This Page

This content is loaded from a Markdown file and rendered using the `content.html.twig` template.

You can customize this content by editing the Markdown file or by creating your own content files.

## Features

- Markdown rendering with Parsedown
- Customizable page metadata via `_pages.php`
- SEO-optimized templates
- Responsive design

## Getting Started

To create your own content:

1. Create a new `.md` file in the `content/` directory
2. Add page metadata to `content/_pages.php`
3. Use the `content.html.twig` template to render it

For more information, see the [documentation](README.md).
